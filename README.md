# 🧠 NeuroSafeAI
**AI-Powered Brain Hemorrhage Detection**

A full-stack AI system that analyzes CT scan images, detects brain hemorrhage using deep learning, and autonomously alerts medical staff when a critical case is found.

Built entirely from scratch — model training, backend API, agentic alert system, and frontend dashboard.

---

## How It Works
Upload a CT scan → AI model analyzes it → risk score returned → autonomous agent fires an alert if critical → result appears on dashboard.

No human needed in the middle.

---

## Tech Stack
- **AI** — EfficientNet-B0, PyTorch, Transfer Learning
- **Backend** — FastAPI, Uvicorn
- **Agent** — Custom autonomous alert system
- **Frontend** — HTML, CSS, JavaScript
